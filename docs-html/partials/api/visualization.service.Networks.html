  • lineSegmentsRGB(layer, edgeCount)

    Generates an Array of RGB pairs to match Vertex pairs. eg. [R1,G1,B1,R2,G2,B2,R1,G1,B1,R3,G3,B3,...Rn,Gn,Bn,Rm,Gm,Bm] such that all color pairs are represented uniquely ie. one half of matrix where array length = (n^2-n)/2 eg. 1 2 3 4 1 x x x == 1-2 1-3 1-4 3 2 x x == 2-3 2-4 + 2 3 x == 3-4 + 1 4 == ((44)-4)0.5 = 6 pairs of colors Explanation: Networks can be visualized with THREE.LineSegments from THREE.BufferGeometry which stores Arrays of Vertex pairs.

    Parameters
    ParamTypeDetails
    layerArray

    An array of features colors eg. restraints

    edgeCountArray

    An array of color RGB Pairs to match Vertex Pairs
