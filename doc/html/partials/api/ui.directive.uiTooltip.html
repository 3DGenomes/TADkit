uiTooltip
directive in module ui

Description

UI Tooltip. use ng-cloak or style="display:none" on tag to ensure hidden onLoad

Usage

as element:
<ui-tooltip>
</ui-tooltip>
as attribute
<ANY ui-tooltip>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Example

<ui-tooltip title="title" contents="contents" styling="styling" style="display:none" /> <div ui-tooltip title="title" contents="contents" styling="styling" style="display:none"> </div>
