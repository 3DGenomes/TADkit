tkComponentTrackBarchart
directive in module TADkit

Description

Generates a d3 Barchart from the supplied data.

Usage

as element:
<tk-component-track-barchart
       type="{Object}"
       title="{Object}"
       settings="{Array}"
       view="{Array}"
       data="{Array}"
       layer="{Array}"
       togglelayer="{function}">
</tk-component-track-barchart>
as attribute
<ANY tk-component-track-barchart
     type="{Object}"
     title="{Object}"
     settings="{Array}"
     view="{Array}"
     data="{Array}"
     layer="{Array}"
     togglelayer="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
typeObject

Track type.

titleObject

Track title.

settingsArray

Track settings.

viewArray

Track view.

dataArray

Track data.

layerArray

Value for toggle in template.

togglelayerfunction

Toggle function in template.
