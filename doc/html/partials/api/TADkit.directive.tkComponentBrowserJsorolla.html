tkComponentBrowserJsorolla
directive in module TADkit

Description

OpenCB/jsorolla/ genome-viewer directive that is replaced on complie by real component directive from supplied object type. e.g. from a array of components objects

Usage

as element:
<tk-component-browser-jsorolla>
</tk-component-browser-jsorolla>
as attribute
<ANY tk-component-browser-jsorolla>
   ...
</ANY>

Example

<div tk-component-track-browser-jsorolla ng-repeat='component in components'> </div>
