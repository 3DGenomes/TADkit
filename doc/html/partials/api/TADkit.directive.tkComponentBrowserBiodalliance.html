tkComponentBrowserBiodalliance
directive in module TADkit

Description

Biodalliance browser directive that is replaced on complie by real component directive from supplied object type. e.g. from a array of components objects

Usage

as element:
<tk-component-browser-biodalliance>
</tk-component-browser-biodalliance>
as attribute
<ANY tk-component-browser-biodalliance>
   ...
</ANY>

Example

<div tk-component-track-biodalliance ng-repeat='component in components'> </div>
